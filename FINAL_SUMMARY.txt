    itemId: ObjectId (ref: Item),
    quantity: Number
  }]
  totalPrice: Number
  status: String
  createdAt: Date

âœ… Password Security

   - Secure comparison on login

âœ… JWT Authentication
   - Tokens generated on successful login
   - Verified on every protected request
   - Stored in database for validation
   - Signed with JWT_SECRET

âœ… Single-Device Session
   - Only one token per user can exist
   - New login blocks if token already exists
   - Error: "You cannot login on another device"
   - Token cleared on logout

âœ… Protected Routes
   - All cart operations require token
   - All order operations require token
   - Middleware validates token against database
   - User verified at request level

âœ… CORS Security
   - CORS configured for frontend URL
   - Prevents unauthorized API access
   - Same-origin policy enforced
